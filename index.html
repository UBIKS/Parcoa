Parcoa: Objective-C Parser Combinators

Parcoa is a collection of parsers and parser combinators for Objective-C inspired by Haskell's Parsec package and Python's Parcon library. It is released under a MIT license.

For an overview of Parcoa's functionality read the project README.

  • Primitive parsers for consuming unicode characters and strings can be found in Parcoa+Primitives.h.
  • Parser Combinators for combining these parsers can be found in Parcoa+Combinators.h.
