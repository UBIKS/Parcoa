Overview

A wrapper around a ParcoaPredicateBlock.

For example, to create a predicate that matches the unichars ‘a’ and ‘b’:

  [ParcoaPredicate predicateWithBlock:^BOOL(unichar x) {
      return x == 'a' || x == 'b';
  } name:@"a|b" summary:@"'a' or 'b'"];
