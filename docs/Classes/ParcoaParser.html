Overview

The fundamental unit of parsing is Parcoa. ParcoaParser is a lite wrapper around a ParcoaParserBlock. A ParcoaParser attempts to parse an input string, returning an OK or Fail result. All parsers within Parcoa are ParcoaParser objects.

  • Parse a NSString using parse:.
  • Create new parsers using the parserWithBlock:name:summary: class method.

The parser combinators in the Parcoa+Combinators category are all pure class methods. This ParcoaParser+Combinators category adds instance methods to easily apply combinators.
