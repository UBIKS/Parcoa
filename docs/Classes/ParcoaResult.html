traceback:full:

Generates a traceback of Parcoa expectations. If full is TRUE then a complete snapshot of the Parcoa expectation is given, if FALSE only the parser branch that consumed the most input is output.

- (NSString *)traceback:(NSString *)input full:(BOOL)full

Parameters

input

The previously supplied parser input.

full

TRUE for all expectations, FALSE for the expectation the consumed the most input.

Return Value

A traceback containing line and column numbers along with details of what the parser expected.

Discussion

Generates a traceback of Parcoa expectations. If full is TRUE then a complete snapshot of the Parcoa expectation is given, if FALSE only the parser branch that consumed the most input is output.
